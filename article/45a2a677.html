前言

很多人会在网上看到Hexo静态博客(网站)搭建教程，如：GitHub Pages、Gitee Pages(码云)、Coding Pages、他们或多或少都有缺点

Github Pages缺点

  1. 访问速度慢
  2. 页面全英文(影响不大)
  3. 不支持多域名绑定
  4. 百度不能收录

Gitee Pages缺点

  1. 必须手动更新部署内容
  2. 不支持自定义域名(付费可免除，手动更新部署、支持自定义域名)

Coding Pages缺点

  1. 网络不稳定

腾讯云静态网站托管 - 缺点

  1. 不能使用git push上传

腾讯云静态网站托管
